# 応援ロケット

## Setup

### Install Volta

```bash
curl https://get.volta.sh | bash
```

### Install Corepack

```bash
volta install corepack
```

### Enable pnpm

```bash
corepack enable
corepack enable pnpm
```

### Install Dependencies

```bash
pnmm i
```

### Start Android Studio

```bash
npx cap open android
```

### Start XCode

```bash
npx cap open ios
```
