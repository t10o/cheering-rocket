# 応援ロケット

## Setup

### Install Volta

```bash
curl https://get.volta.sh | bash
```

### Install Corepack

```bash
volta install corepack
```

### Enable pnpm

```bash
corepack enable
corepack enable pnpm
```

### Install Dependencies

```bash
pnmm i
```

### Install lonic CLI

```bash
npm install -g @ionic/cli native-run
```

### Start Android

```bash
npx cap open android
```

### Start iOS

```bash
npx cap open ios
```
